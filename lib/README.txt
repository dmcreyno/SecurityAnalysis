This folder will contain libraries that are needed to build this module.
The pom for this module assumes you have installed the libraries in your
local maven repo. If you elect not to install these libraries in your
local maven repo you will need to edit the pom to reference local files.
You are encouraged to install the libraries in your local maven repo as
currently configured in the pom.

Libraries:
  f2jutils.jar
          	<groupId>org.netlib</groupId>
        	<artifactId>f2jutil</artifactId>
        	<version>1.0</version>
